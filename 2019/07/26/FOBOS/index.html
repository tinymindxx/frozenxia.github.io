FOBOS

Posted on 2019-07-26 11:18   |   In algorithm   |   views

FOBOS(Foward Backward Spliting)算法的更新规则分为两个阶段:
$$w_{t+\frac{1}{2}}=w_{t}-\eta_{t}g_{t}^{f}\tag{1}$$

$$w_{t+1}=\underset{w}{argmin} \left \{\frac{1}{2}||w-w_{t+\frac{1}{2}}||^2 + \eta_{t+\frac{1}{2}}r(w) \right \}\tag{2}$$
第一阶段就是标准的SGD过程,第二个过程是对第一阶段得到的值进行修正.仔细观察第二个阶段,也可以分为两部分,前一部分控制微调的结果保持在梯度下降附近,后一部分是正则化.如果将两个公式(1),(2)合并之后,有:
$$w_{t+1}=\underset{w}{argmin} \left \{\frac{1}{2}||w- w_{t}+\eta_{t}g_{t}^{f}||^2 + \eta_{t+\frac{1}{2}}r(w) \right \}\tag{3}$$
令$F(w)= \frac{1}{2}||w- w_{t}+\eta_{t}g_{t}^{f}||^2 + \eta_{t+\frac{1}{2}}r(w) $,如果$w_{t+1}$存在最优解,那么$0$一定属于$F(w)$的次梯度集合:
$$0 \in \partial F(w)= w- w_{t}+\eta_{t}g_{t}^{f} + \eta_{t+\frac{1}{2}} \partial r(w)$$
由于$w_{t+1}=\underset{w}{argmin}F(w)$,所以:
$$0= \left \{w- w_{t}+\eta_{t}g_{t}^{f} + \eta_{t+\frac{1}{2}} \partial r(w) \right \}\bigg | _{w_{t+1}}$$
代入$w_{t+1}$有:
$$w_{t+1}= w_{t}-\eta_{t}g_{t}^{f} - \eta_{t+\frac{1}{2}} \partial r(w_{t+1}) $$
上面的等式可以看出,$w_{t+1}$与$w_{t}$是线性关系,因此增加的计算成本可以忽略不计.
对上面的等式作一些简单变换,令$v$表示$w_{t+\frac{1}{2}}=w_{t}-\eta_{t}g_{t}^{f}$,$\tilde{\lambda}$表示$\eta_{t+\frac{1}{2}}.\lambda$.那么等式(3)可以表示成:
$$\underset{w}{minimize} \left \{\frac{1}{2}||w- v||^2 + \tilde{\lambda}r(w) \right \} \tag{4}$$

FOBOS with $\mathcal{l_1}$ Regularization

对于$\mathcal{l_1}$ Regularization,$r(w)=\lambda |w |_1$,对等式(4)进行重写之后,有:
$$\underset{w}{minimize} \sum_{j=1}^{n} \left (\frac{1}{2}||w_j- v_j||^2 + \tilde{\lambda}|w_j| \right ) \tag{5}$$

等式(5)主要关注$w$中第$j$个参数,简单起见省略坐标$j$,用$w^*$表示使得$\frac{1}{2}||w- v||^2 + \tilde{\lambda}|w|$最小的参数.很明显,$w^*.v \geq 0$.如果$w^*.v < 0$,那么$\frac{1}{2}v^2 < \frac{1}{2}v^2 - w^*.v + \frac{1}{2}{(w^*)}^2 < \frac{1}{2}(w- v)^2 + \tilde{\lambda}|w| $,根据定义,$w^*$是最优解,因此矛盾.
有$w^*.v \geq 0$,分两种情况讨论:

  1. 假设$v \geq 0$,目标函数变成$\frac{1}{2}(w- v)^2 + \tilde{\lambda}w$,其中$w\geq 0 $,相当于引入了限制条件$-w\leq 0 $.引入拉格朗日式子$\beta \geq 0$,得到$\frac{1}{2}(w- v)^2 + \tilde{\lambda}w-\beta w$,根据KKT条件,有:$\frac{\partial}{\partial w}\left (\frac{1}{2}(w-v)^2 + \tilde{\lambda}w -\beta w \right ) = 0$,以及$\beta w = 0$,从而得到:$w^*=v-\tilde{\lambda}+\beta$.

    • 如果$w^* >0 $,那么有$\beta=0$,因此$w^*=v-\tilde{\lambda}$,由于$w^* >0 $,可得$v-\tilde{\lambda} >0$;
    • 如果$w^* =0 $,那么有$0 =v-\tilde{\lambda}+\beta$,且$\beta \leq 0$,所以$v-\tilde{\lambda} <0$
      综合上述两种情况,得到在$v \geq 0$条件下,$w^*=max(0,v-\tilde{\lambda})$

  2. 同理在$v < 0$时,可得$w^*=-max(0,-v-\tilde{\lambda})$

综上所述,FOBOS在L1正则化条件下,权重更新的方法为:
$$
w^* = sign(v)max(0,|v|-\tilde{\lambda})=sign(w_{t}-\eta_{t}g_{t}^{f})max(0,|w_{t}-\eta_{t}g_{t}^{f}|-\eta_{t+\frac{1}{2}}.\lambda) \tag{6}
$$

FOBOS-$L_2^2$

当$r(w)=\frac{\lambda}{2}|w|_2^2$时,优化函数变成:
$$\underset{w}{minimize} \sum_{j=1}^{n} \left (\frac{1}{2}||w_j- v_j||^2 + \frac{1}{2}\tilde{\lambda}|w_j|_2^2 \right ) \tag{7}$$
求偏导数之后可以得到$w_j^*-v_j+\tilde{\lambda}w_j^*=0$,所以:
$$
w_j^*=\frac{w_{t}-\eta_{t}g_{t}^{f}}{1+\tilde{\lambda}}
$$
也就是每次更新之后对$w_{t+1}$的值仅仅作简单的放缩处理.

FOBOS-L1 与TG的关系

从公式$(6)$中可以看出,当满足$|w_{t}-\eta_{t}g_{t}^{f}| \leq \eta_{t+\frac{1}{2}}.\lambda$时,$w^*$会被截断成$0$,也就是说如果当一条样本产生的梯度不够对权重发生足够大的变化($\eta_{t+\frac{1}{2}}.\lambda$),那么本次更新权重中它就不重要,其权重会变成0
同时,对$6$进行重写之后,得到如下公式:
$$
w^*=\begin{cases}0 &if \, |w_{t}-\eta_{t}g_{t}^{f}| \leq \eta_{t+\frac{1}{2}}.\lambda \cr sign(w_{t}-\eta_{t}g_{t}^{f})(|w_{t}-\eta_{t}g_{t}^{f}|-\eta_{t+\frac{1}{2}}.\lambda) & otherwise\end{cases}
$$
进一步简化之后得到:
$$
w^*=\begin{cases}0 &if \, |w_{t}-\eta_{t}g_{t}^{f}| \leq \eta_{t+\frac{1}{2}}.\lambda \cr (w_{t}-\eta_{t}g_{t}^{f})- \eta_{t+\frac{1}{2}}.\lambda sign(w_{t}-\eta_{t}g_{t}^{f}) & otherwise\end{cases}
$$
对比Truncate gradient,如果令$\theta=\infty,k=1,\lambda=\eta_{t+\frac{1}{2}}.\lambda$,那么FOBOS-L1与Truncate Gradient变成相同的更新公式了.所以,FOBOS-L1可以被认为是Truncate Gradient的一种特殊情况.
